# 肺がん早期発見プロジェクト

- CTスキャン画像を用いた、肺の悪性腫瘍の検出です。

## データ
- [LUNAグランドチャレンジ](https://luna16.grand-challenge.org/download/)のデータを用います。
- `candidates.csv`
  - 結節の候補に関するデータ
  - class列は0が結節ではない、1が悪性・良性問わず結節の候補
- `annotations.csv`
    - 結節候補に関する他のデータ
